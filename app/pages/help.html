IP Camera Setup

(use Mobile Phone as IP Camera)

  1. Install an IP camera app (e.g., IP Webcam, DroidCam for Android) on your phone.
  2. Connect your phone and computer to the same Wi-Fi network.
  3. Start the IP camera app and note the stream URL (e.g., http://192.168.1.100:8080).
  4. In OpenLens, select IP Camera and enter the URL (use /shot.jpg for snapshots or /video for MJPEG stream).
  5. Click Start to view and analyze the stream.

Voice Commands

  • Use "Microphone" button to activate voice commands (e.g., "capture image", "pause video").
  • Supported commands: "capture image", "pause video", "resume video", "save video", "record video", etc.

List of Controls

  • Play: Start video playback.
  • Pause: Pause video playback.
  • Resume: Resume video playback.
  • Capture: Take a snapshot of the current video frame.
  • Save: Save the recorded video.
  • Record: Start video recording.
  • History: View the history and record data.
  • Exit: Exit the app.

Troubleshooting

  • Camera not detected: Ensure permissions are granted for supported browsers (Chrome, Edge, or Safari).
  • IP Camera not working: Double-check the URL, network connection, and IP camera app is running on your phone.
  • Voice commands not working: Make sure your microphone is enabled and permissions are granted.
