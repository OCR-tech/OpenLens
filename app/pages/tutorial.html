Tutorial

Getting Started

  1. Start Web App:
    • Navigate to the directory app/web.
    • Run a local server: python -m http.server 8000.
    • Open App in your browser: http://localhost:8000/ or http://<ip-address>:8000/ on your network.
  2. Select Video Source:
    • Camera (default): Uses your device's built-in webcam.
    • USB Camera: Uses an external USB camera.
    • IP Camera: Enter the URL of an IP camera (e.g., from an IP camera app running on a mobile phone).
    • Video File: Upload a video file for detection.
  3. Grant Permissions: Allow camera and microphone access when prompted.
  4. Click "Start": Start the app for real-time object detection.

How to Use

  1. Object Detection:
    • "Objects": Show or hide labels and bounding boxes of detected objects.
    • "Labels": Show or hide labels of detected objects.
    • "Boxes": Show or hide bounding boxes of detected objects.
  2. Voice Commands:
    • "Microphone": Activate voice commands (e.g., "capture image", "pause video").
  3. Save or Capture:
    • "Capture": Take snapshots of the current video frame.
    • "Save": Save video segments after stopping the recording.
